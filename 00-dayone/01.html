NOTES

  • When using emmet, class is notated by a "." and id is by a "#"
  • HTML stands for Hypertext Markup Language
  • DOCTYPE states the type pf our document. In this case, HTML
  • our html tag denotes our language to be English, per the land="en"
  • The head tag houses metdata, or detailed data about our doc
  • UTF-8 specifies character encoding. Each character (letter, number, even spaces) has a numeric value
  • Viewport notes the device we are reading our document on.
  • Title reflects the name that displays on our browser tab.
  • A style tag can help us inject css right into our document. *Not the best practice
  • Link allows us to connect to other files, such as our stylesheet
  • The body tag houses all the other tags that we want to display
